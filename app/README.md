# Task App Client side

タスク管理アプリのクライアントサイドアプリケーション

## Summary

- React
- Vite
- Bun
- Tanstack Router
- shadcn/ui
- tailwindcss
- react-hook-form
- openapi-fetch

## Getting Started

Bunをインストール

https://bun.sh/docs/installation

`.env.example`を参考に`.env`を作成

開発サーバーを起動

```bash
bun i   # 依存関係をインストール
bun dev # 開発サーバーを起動
```
